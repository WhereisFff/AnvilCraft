# 量产绿宝石

- 拥有[<translate key="block.anvilcraft.giant_anvil"/>](../005_block/210_giant_anvil.md)后，可以[制造刷怪笼](210_spawner.md)
- 获得卫道士刷怪笼
- 击杀卫道士获得<translate key="item.minecraft.emerald"/>

# 自动化

- 尽管卫道士只有被玩家击杀，才会掉落<translate key="item.minecraft.emerald"/>
- 我们仍可以通过[<translate key="block.anvilcraft.ember_anvil"/>](../005_block/223_ember_anvil.md)对其进行[铁砧抢夺](../001_feature/000_anvil_loot.md)，自动化生产<translate key="item.minecraft.emerald"/>
