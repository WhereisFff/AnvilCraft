# 加工：宝石转化机

# 前言

- 阅读[宝石转化](../007_recipe/204_gem.md)可知其挑战性
- 又是丢宝石块，又是丢宝石，还要换掉锅下的工作方块，还要砸锅，估计光是想想就让你感到头疼了
- 我们不妨打开思路，看看下面的示例机器，你一定会从中有所收获

这台机器通过推拉锅来不断切换下方的工作方块，同时使用<translate key="item.minecraft.smooth_stone"/>砸锅加工。复杂精妙的时序使得加工速度很高。

- 所有 <translate key="item.minecraft.smooth_stone"/> 可替换为 任意完整不透明方块
- 所有 <translate key="item.minecraft.copper_bulb"/> 可替换为 任意铜灯
- 确保 <translate key="item.minecraft.observer"/> 方向正确  
- 使用：
  - 拉下拉杆关机
  - 填入原料后，复位拉杆运行
  - 运行过程中不可关机！
