# 加工：宝石转化机

- 阅读[宝石转化](../008_recipe/204_gem.md)可知其挑战性
- 又是丢宝石块，又是丢宝石，还要换掉锅下的工作方块，还要砸锅，估计光是想想就让你感到头疼了
- 我们不妨打开思路，看看下面的示例机器，你一定会从中有所收获

这台机器通过推拉锅来不断切换下方的工作方块，同时使用<ref item="minecraft:smooth_stone"/>砸锅加工。复杂精妙的时序使得加工速度很高。

## 图示

<structure id="../../structures/machine/gem.snbt"/>

<tip>
左键调整位置；右键调整角度；PgUP/PgDN调整显示高度
</tip>

1. 使用按钮修改铜灯，使其默认为亮起状态（之后可以把按钮拆了）
2. 使用铁砧锤右键炼药锅正面，创建输出槽
3. *比较器*朝向的*投掷器*放入绿宝石块
4. 更远的*投掷器*放入等量宝石
5. 使用：
  - 填入原料后，复位拉杆运行
  - 拉下拉杆关机

<warning>
运行过程中不可关机！
</warning>

- 所有 <ref item="minecraft:smooth_stone"/> 可替换为 任意完整不透明方块
- 所有 <ref item="minecraft:copper_bulb"/> 可替换为 任意铜灯
- 确保 <ref item="minecraft:observer"/> 方向正确
