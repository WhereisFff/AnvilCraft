# 糖块

<recipe id="anvilcraft:sugar_block"/>

# 功能

- 被铁砧砸击时，产生1个[电荷](../003_power/100_basic_power_generation.md#电荷)
- 被砸时有 5% 概率碎裂一次，4次以后碎为9个糖

# 特性

- 木镐可挖掘
- 挖掘掉落自身
