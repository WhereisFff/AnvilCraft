# 推进活塞

<recipe id="anvilcraft:propel_piston"/>

# 功能

- 受到红石信号或被玩家右键后，持续向前移动并推动方块
- 工作耗电量与被推动的方块数量成正比
- 手动右键停机、能量耗尽或遇到推不动的方块时，停止工作
- 充能：
  - 背部接收到激光时可以充能，充能功率 = 激光等级 * 15kW
  - 可以使用电容手动充能
