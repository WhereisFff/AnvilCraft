# 燧石块

<recipe id="anvilcraft:flint_block"/>

# 功能

- 可被切石为建筑方块
- 若<translate key="block.minecraft.iron_block"/>和[<translate key="block.anvilcraft.flint_block"/>](009_flint_block.md)因<translate key="item.minecraft.piston"/>推动而摩擦，
  则在推动后以被推动的方块为中心，点燃一次3*3*3范围内的油锅和营火；若二者不存在则会燃起火焰

# 特性

- 木镐可挖掘
- 挖掘掉落自身
