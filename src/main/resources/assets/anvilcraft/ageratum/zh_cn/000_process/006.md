# 6-铸造皇家钢

# 铸造

生产[<translate key="item.anvilcraft.royal_steel_ingot"/>](../002_material/110_royal_steel.md)是每个砧人的成年礼

# 使用

1. 和[珠宝匠](../001_feature/001_jeweler.md)交易 ~~(或者去抢劫铁匠铺的箱子)~~ 获得[<translate key="item.anvilcraft.royal_steel_upgrade_smithing_template"/>](../002_material/109_royal_template.md)
2. 制作[<translate key="block.anvilcraft.royal_smithing_table"/>](../004_block/101_royal_smithing_table.md)，它可以不消耗锻造模板制作装备
3. 制作[<translate key="block.anvilcraft.royal_anvil"/>](../004_block/103_royal_anvil.md)和[<translate key="block.anvilcraft.royal_grindstone"/>](../004_block/102_royal_grindstone.md)
4. 使用[<translate key="item.anvilcraft.royal_steel_ingot"/>](../002_material/110_royal_steel.md)将各种工具升级
