# 6-铸造皇家钢

# 铸造

生产<ref item="anvilcraft:royal_steel_ingot"/>是每个砧人的成年礼

# 使用

1. 和[珠宝匠](../001_feature/001_jeweler.md)交易 ~~(或者去抢劫铁匠铺的箱子)~~ 获得<ref item="anvilcraft:royal_steel_upgrade_smithing_template"/>
2. 制作<ref item="anvilcraft:royal_smithing_table"/>，它可以不消耗锻造模板制作装备
3. 制作<ref item="anvilcraft:royal_anvil"/>和<ref item="anvilcraft:royal_grindstone"/>
4. 使用<ref item="anvilcraft:royal_steel_ingot"/>将各种工具升级
