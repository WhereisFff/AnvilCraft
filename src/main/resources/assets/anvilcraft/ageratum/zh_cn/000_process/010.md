# 10-冰与火之歌

# <translate key="item.anvilcraft.frost_metal_ingot"/>

## 合成

1. 获得一些细雪，试试依靠[本模组提供的方式](../008_recipe/110_ice.md)
2. 生产一些[<translate key="item.anvilcraft.frost_metal_ingot"/>](../002_material/202_frost_metal.md)

## 使用

- 合成并尝试浮霜金属工具，它们可以将附魔转化为纯粹的数值
- 制作[浮霜锻造台](../004_block/231_frost_smithing_table.md)、[浮霜砂轮](../004_block/232_frost_grindstone.md)和[浮霜铁砧](../004_block/233_frost_anvil.md)

# <translate key="item.anvilcraft.ember_metal_ingot"/>

## 合成

1. [生产一些下界合金](../008_recipe/210_netherite_ingot.md)
2. 生产[<translate key="item.anvilcraft.oil_bucket"/>](../002_material/201_oil.md)，获得*燃烧的炼药锅*
3. 生产一些[<translate key="item.anvilcraft.ember_metal_ingot"/>](../002_material/211_ember_metal.md)

## 使用

- 合成并尝试余烬金属工具，它们可以凭借火的力量自我修复
- 制作[<translate key="block.anvilcraft.ember_anvil"/>](../004_block/223_ember_anvil.md), [<translate key="block.anvilcraft.ember_grindstone"/>](../004_block/222_ember_grindstone.md), [<translate key="block.anvilcraft.ember_smithing_table"/>](../004_block/221_ember_smithing_table.md)
