# 10-冰火试炼

# <ref item="anvilcraft:frost_metal_ingot"/>

## 合成

1. 获得一些细雪，试试依靠[本模组提供的方式](../008_recipe/110_ice.md)
2. 生产一些<ref item="anvilcraft:frost_metal_ingot"/>

## 使用

- 合成并尝试浮霜金属工具，它们可以将附魔转化为纯粹的数值
- 制作[浮霜锻造台](../004_block/231_frost_smithing_table.md)、<ref item="anvilcraft:frost_grindstone"/>和<ref item="anvilcraft:frost_anvil"/>
- 利用<ref item="anvilcraft:frost_anvil"/>的特性刷取经验球，然后用<ref item="anvilcraft:exp_collector"/>收集，再固化为<ref item="anvilcraft:exp_gem"/>，给自己或村民使用

<tip>
对于液态经验的获取，可以采取前文使用<ref item="anvilcraft:frost_anvil"/>的特性，亦可以制造刷怪塔等传统方式获得大量经验球，再用<ref item="anvilcraft:exp_collector"/>收集
</tip>

# <ref item="anvilcraft:ember_metal_ingot"/>

## 合成

1. [生产一些下界合金](../008_recipe/210_netherite_ingot.md)
2. 生产<ref item="anvilcraft:oil_bucket"/>，获得*燃烧的炼药锅*
3. 生产一些<ref item="anvilcraft:ember_metal_ingot"/>

## 使用

- 合成并尝试余烬金属工具，它们可以凭借火的力量自我修复
- 制作<ref item="anvilcraft:ember_anvil"/>, <ref item="anvilcraft:ember_grindstone"/>, <ref item="anvilcraft:ember_smithing_table"/>
- 制作凋灵笼子：余烬金属不会被凋灵破坏，可以困住凋灵，便于安全地击杀它

# 支线：大闹试炼厅

- 通过前两种材料获得强大的装备，然后进入试炼厅获得材料
- 本模组提供了[很多便利](../008_recipe/205_mace.md)
- 最终，你可以[量产重锤](../008_recipe/205_mace.md)
