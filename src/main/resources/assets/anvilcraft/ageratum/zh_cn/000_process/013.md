# 13-新物质

# 中子锭

1. 制作[<translate key="block.anvilcraft.space_overcompressor"/>](../004_block/220_space_overcompressor.md)，这需要8个<translate key="item.minecraft.shulker_shell"/>
2. 准备好大量[<translate key="block.anvilcraft.heavy_iron_block"/>](../002_material/007_heavy_iron_block.md)(或其他金属)后，将其压入[<translate key="block.anvilcraft.space_overcompressor"/>](../004_block/220_space_overcompressor.md)中，即可合成[<translate key="item.anvilcraft.neutronium_ingot"/>](../002_material/221_neutronium_ingot.md)

<tip>
可以[制作潜影贝刷怪笼](../008_recipe/210_spawner.md)来获取大量<translate key="item.minecraft.shulker_shell"/>，由于潜影贝免疫铁砧，可以尝试通过[<translate key="block.anvilcraft.tesla_tower"/>](../004_block/201_tesla_tower.md)处死。
</tip>

<tip>
极其耗铁，建议对刷铁机大升级，比如多加些修复发射器和铁傀儡
</tip>

# 钚

2. 目前[钚](../002_material/321_plutonium.md)主要用来发电
