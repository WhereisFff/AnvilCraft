# 8-巨型铁砧

# 获得<ref item="anvilcraft:giant_anvil"/>

通过<ref item="anvilcraft:corrupted_beacon"/>生产<ref item="anvilcraft:giant_anvil"/>

## 树厂与农场

1. 发挥[<ref item="anvilcraft:giant_anvil"/>的撼地能力](../004_block/210_giant_anvil.md)，启用破坏模式，自动化砍伐树木和收割农作物。农作物会自动补种。
2. 使用<ref item="anvilcraft:item_collector"/>收集大范围的掉落物
3. 使用<ref item="anvilcraft:block_placer"/>或<ref item="anvilcraft:smart_block_placer"/> 种植树苗
4. 可以使用<ref item="anvilcraft:induction_light"/>催熟树苗

<tip>
手册提供了[示例机器](../009_machine/210_tree.md)，可以根据需求自行调整
</tip>

## 量产钻石

利用富含碳元素的原木，可以[自动化量产<ref item="minecraft:diamond"/>](../008_recipe/201_diamond.md)！
