# 流程5-迈入电气时代

# 发电

制造[<translate key="block.anvilcraft.charge_collector"/>](../003_power/100_basic_power_generation.md)，通过本模组的特殊方法发电

# 输电

- 制造[<translate key="block.anvilcraft.transmission_pole"/>](../003_power/101_power_transmission.md)，它们可以创建巨大的电网
- 电网可以将附近所有电力设施连接在一起
- 将[<translate key="item.anvilcraft.anvil_hammer"/>](../004_tool/000_anvil_hammer.md)戴在头上，就可以看到电网

# 用电

- 学习使用[<translate key="block.anvilcraft.heater"/>](../005_block/100_heater.md)进行高温熔炼
- 使用[<translate key="block.anvilcraft.charger"/>](../003_power/102_power_charge.md)制作[<translate key="item.anvilcraft.magnet_ingot"/>](../002_material/001_magnet.md)
- 使用[<translate key="item.anvilcraft.ionocraft_backpack"/>](../004_tool/101_ionocraft_backpack.md)尽情飞翔
- 使用[<translate key="block.anvilcraft.induction_light"/>](../005_block/101_induction_light.md)催生农作物
