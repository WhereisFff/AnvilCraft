# 5-迈入电气时代

# 发电

- (可选)了解本模组提供的[更强的红石元件](../004_block/001_redstone.md)
- 制造<ref item="anvilcraft:charge_collector"/>，通过本模组的特殊方法发电

<tip>
可以参考手册内置的[简易发电机](../009_machine/101_piezoelectric_crystal.md)
</tip>

# 输电

- 制造<ref item="anvilcraft:transmission_pole"/>，它们可以创建巨大的电网
- 电网可以将附近所有电力设施连接在一起
- 将<ref item="anvilcraft:anvil_hammer"/>戴在头上，就可以看到电网

# 用电

- 学习使用<ref item="anvilcraft:heater"/>进行高温熔炼，并将[宝石融化](../002_material/106_chromatic_stone.md)
- 使用<ref item="anvilcraft:charger"/>制作<ref item="anvilcraft:magnet_ingot"/>
- 使用<ref item="anvilcraft:ionocraft_backpack"/>尽情飞翔
- 使用<ref item="anvilcraft:induction_light"/>催生农作物
- 使用[<ref item="anvilcraft:batch_cutter"/>/<ref item="anvilcraft:batch_crafter"/>](../004_block/101_batch_craft.md)自动化制作

# 流体系统
- 流体将参与合成，届时需要用到本模组受重力影响的[流体系统](../004_block/110_fluid_system.md)
