# 3-学习铁砧加工

<info> 
蓝色字体可以点击，前往关联页面 
</info>

# 制造<ref item="anvilcraft:anvil_hammer"/>

通过<ref item="anvilcraft:anvil_hammer"/>，可以快速拆卸本模组的大部分方块，值得合成

# 铁砧加工

- <ref item="minecraft:anvil"/>落在特定的方块上时，可以加工该方块，完成[方块加工](../007_struct/000_block_processing.md)
- <ref item="minecraft:anvil"/>落在特定的方块上时，会加工其中或其上的物品，完成[物品加工](../007_struct/000_item_processing.md)

<warning>
这些内容较为重要，请务必全部掌握
</warning>

- 使用<ref item="anvilcraft:magnet_block"/>吸引<ref item="minecraft:anvil"/>，再用红石信号使<ref item="anvilcraft:magnet_block"/>失去磁性，使铁砧下落
- 可以使用<ref item="anvilcraft:fish_tank"/>替代<ref item="minecraft:cauldron"/>，二者各有优点
- 前期资源紧张，可以试试[合成破铁砧](../001_feature/000_anvil_fix.md)节省资源，后续再使用[铁砧修补术](../001_feature/000_anvil_fix.md)

<tip>
若没有<ref item="anvilcraft:magnet_block"/>，也可以手动拆除铁砧再将其放置或者设计红石机器
</tip>

<tip>
使用<ref item="anvilcraft:anvil_hammer"/>右键<ref item="minecraft:cauldron"/>，可以生成炼药锅输出槽，自动将产物导出，参考[炼药锅增强](../001_feature/000_cauldron.md)
</tip>
