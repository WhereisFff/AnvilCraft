# 树脂块

<item id="anvilcraft:resin_block"/>

# 捕捉生物
- 手持<ref item="anvilcraft:resin_block"/>，右键**碰撞箱**的长宽小于1.5,高小于2的生物，可以进行抓捕
- 但抓捕敌对和中立生物这要求其受到**虚弱效果**
- 可以右键地面释放生物。

<info>
eg:村民可以抓取;僵尸需要对其施加虚弱效果才能抓取;铁傀儡不能抓取
</info>

- 可以通过<ref item="minecraft:dispenser"/>自动化操作
