# 晶洞

<item id="anvilcraft:geode"/>


# 获取方式

- 挖掘<ref item="minecraft:budding_amethyst"/>
- 与[珠宝匠](../001_feature/001_jeweler.md)交易
- 开局的奖励箱(需要在创建世界时开启奖励箱选项)

# 功能

- 寻找附近的紫水晶洞

# 参与合成

- 晶洞在[物品冲压](../007_struct/000_block_processing.md)时，一定出4个<ref item="minecraft:amethyst_shard"/>，概率出[黄玉、蓝宝石和红宝石](../002_material/000_gems.md)
- 对晶洞进行时移，可以合成出<ref item="minecraft:budding_amethyst"/>
