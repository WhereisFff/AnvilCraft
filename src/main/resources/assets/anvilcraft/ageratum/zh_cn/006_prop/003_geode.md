# 晶洞

<item id="anvilcraft:geode"/>


# 获取方式

- 挖掘紫水晶母岩(时运无法生效)
- 与[珠宝匠](../001_feature/001_jeweler.md)交易
- 开局的奖励箱(需要在创建世界时开启奖励箱选项)

# 功能

- 使用时能够找到附近的紫水晶洞(地物)
- 参与合成

# 参与合成

- 在晶洞在[物品冲压](../007_struct/000_block_processing.md)时，一定出4个紫水晶碎片，概率出[黄玉、蓝宝石和红宝石](../002_material/000_gems.md)
- 对晶洞进行时移，可以合成出紫水晶母岩
