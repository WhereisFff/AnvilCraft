# 使用

- 装填：副手持武器，主手持幻灵弹弓，右键装填
- 待机：与原版弩类似，鼠标指上去可以看到说明中有弹药信息
- 射击：点按右键射出弹药，长按右键连续射击，每次射击之间冷却为2秒
  - 射击不会消耗武器耐久，而是射出不受重力影响的幻影
- 取出和替换弹药：另一只手为空时shift右键取出弹药，另一只手为与弹药不同的物品时shift右键替换弹药

# 特性

- 伤害：该武器近战伤害的50%
  - 原武器仅增伤附魔生效，无情时增伤附魔不生效，然后可以通过幻灵弹弓的力量附魔增伤
- 附魔说明：
  - 快速装填：影响装填速度，与原版弩一致，同时降低冷却，每级降低0.25秒，最多降1秒
  - 无限：装填装填时不消耗手中的物品，但取出或替换弹药时也不会从中获得物品
  - 力量：增加弹射物伤害（与原版箭矢不同，每级只增加10%伤害）
  - 其余和原版一致
