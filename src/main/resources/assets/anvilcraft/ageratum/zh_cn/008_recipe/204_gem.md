# 量产绿宝石

- 拥有<ref item="anvilcraft:giant_anvil"/>后，可以[制造刷怪笼](210_spawner.md)
- 获得卫道士刷怪笼
- 击杀卫道士获得<ref item="minecraft:emerald"/>

# 自动化

- 尽管卫道士只有被玩家击杀，才会掉落<ref item="minecraft:emerald"/>
- 我们仍可以通过<ref item="anvilcraft:ember_anvil"/>对其进行[铁砧抢夺](../001_feature/000_anvil_loot.md)，自动化生产<ref item="minecraft:emerald"/>
