# 量产<ref item="minecraft:trial_key"/>

进入试炼厅，挑战其中的怪物，获得丰厚的奖励，包括<ref item="minecraft:trial_key"/>，然后可以通过<ref item="anvilcraft:jewelcrafting_table"/>复制

<recipe id="anvilcraft:jewel_crafting/trial_key"/>

# 重置宝库与量产试炼奖励

<row halign="center">
<item id="minecraft:vault"/>
</row>

使用<ref item="anvilcraft:lead_block"/>让<ref item="minecraft:vault"/>忘记自己曾被打开

<structure id="../structures/vault.snbt"/>

然后对着一个<ref item="minecraft:vault"/>重复使用<ref item="minecraft:trial_key"/>，获得大量奖励，有机会获得<ref item="minecraft:ominous_bottle"/>

# 量产<ref item="minecraft:ominous_bottle"/>

这一步可以通过掠夺者前哨站完成，但<ref item="anvilcraft:jewelcrafting_table"/>提供了更方便的复制方法

<recipe id="anvilcraft:jewel_crafting/ominous_bottle"/>

<tip>
其需要的<ref item="minecraft:experience_bottle"/>可以通过用玻璃瓶从装有[经验流体](../002_material/004_exp_gem.md)炼药锅或鱼缸中舀取
</tip>

# 量产<ref item="minecraft:ominous_trial_key"/>

- 服用<ref item="minecraft:ominous_bottle"/>，使得试炼刷怪笼进入高难模式，挑战更强的怪物，获得更丰厚的奖励，包括<ref item="minecraft:ominous_trial_key"/>
- <ref item="minecraft:ominous_trial_key"/>同样可以通过<ref item="anvilcraft:jewelcrafting_table"/>复制量产

<recipe id="anvilcraft:jewel_crafting/ominous_trial_key"/>

使用<ref item="minecraft:ominous_trial_key"/>打开对应等级的<ref item="minecraft:vault"/>，盖绿获得<ref item="minecraft:heavy_core"/>
