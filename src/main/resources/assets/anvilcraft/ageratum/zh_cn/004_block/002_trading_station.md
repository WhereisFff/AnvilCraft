# <ref item="anvilcraft:trading_station"/>

<recipe id="anvilcraft:trading_station"/>

# 功能

1. **GUI左侧交易栏**中，左侧格子放置自己愿意换出的资源，右侧格子放置自己希望收购的资源
2. 在**交易栏下方左侧按钮**，设置*其他玩家*(已完成)或*村民*(未完成)是否可以使用
3. 在**交易栏下方右侧按钮**，设置是否允许自动输入或输出
4. **其他玩家**手持数量足够的**拥有者**希望收购的物品，右键可以交易一次

# 安全机制

- 只有拥有者可以打开
- 被他人 破坏/在周围放置方块 时，会给拥有者发送报警信息
- 如果被非玩家破坏，信息中还包含当时在线的玩家id与最近的玩家id

# 村民贸易（未完成）

村民在工作后会尝试与符合自己交易列表的交易站交互，只要村民可以**接受**交易站的出价

<info>
已知村民收购10黏土球售价1绿宝石，如果交易站提供大于等于10黏土球，需求小于等于1绿宝石，就可以成交
</info>
