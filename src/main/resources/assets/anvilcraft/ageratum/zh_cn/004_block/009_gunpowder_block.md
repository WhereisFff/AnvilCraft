# 火药块

<recipe id="anvilcraft:gunpowder_block"/>

# 功能

- 被铁砧(除<ref item="anvilcraft:giant_anvil"/>外)砸击，或被<ref item="minecraft:flint_and_steel"/>、<ref item="minecraft:fire_charge"/>点燃时爆炸
- 爆炸产生的冲击波仅造成击退效果，并将砸在上方的铁砧推回原位

# 特性

- 木镐可挖掘
- 挖掘掉落自身
