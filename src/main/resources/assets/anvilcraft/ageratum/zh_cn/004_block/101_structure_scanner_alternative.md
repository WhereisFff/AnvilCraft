# 结构扫描仪

<recipe id="anvilcraft:structure_scanner"/>

可以扫描自定义范围内的结构

## 手动保存

打开GUI：
1. 点击右侧按钮扫描结构
2. 放入结构磁盘
3. （可选）输入结构名字
4. 点击左侧的确定按钮保存

## 自动保存

1. 结构磁盘可以通过溜槽等物流方块输入输出
2. 收到红石信号执行保存
