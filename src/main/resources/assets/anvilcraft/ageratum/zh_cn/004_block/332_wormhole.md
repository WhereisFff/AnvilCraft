# 虫洞

- 创建自<ref item="anvilcraft:celestial_forging_anvil"/>
- 所有拥有*虫洞*的[全同黑洞](331_celestial_forging_anvil.md#全同天体)会相互连接

# 加载区块

- 被连接的<ref item="anvilcraft:celestial_forging_anvil"/>只要有一个被加载，则都会被加载
- 以<ref item="anvilcraft:celestial_forging_anvil"/>本体底部中心块为中心3x3区块强加载

# 物流传输

- 被连接的<ref item="anvilcraft:celestial_forging_anvil"/>，周围的同一个位置，只能放上相同种类的[接口](331_celestial_forging_anvil.md#使用接口)
- 此时，同位接口之间共享存储，拥有无线传输的功能

# 实体传送

## 通过<ref item="anvilcraft:celestial_forging_anvil_portal"/>

<recipe id="anvilcraft:celestial_forging_anvil_portal"/>  

- 在同一组全同黑洞中，最多只能放置两个传送门
- 进入传送门，就能从另一个传送门走出

## 通过<ref item="minecraft:ender_pearl"/>

被投掷的<ref item="minecraft:ender_pearl"/>可以进入黑洞，然后随机传送到一个连接的黑洞位置
