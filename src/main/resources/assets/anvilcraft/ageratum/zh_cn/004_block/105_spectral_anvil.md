# 特性
- 不会受到重力影响
- 可被活塞推动
- 使用时，不会损坏
- 当头顶的<ref item="anvilcraft:magnet_block"/>消失或磁性消失时，会向下掉落一个**虚影**
- **虚影**无论从多高砸到一个物体，都相当于从2格高下落的铁砧

# 与铁砧的不同之处
- 无法进行[铁砧抢劫](../001_feature/000_anvil_loot.md)
- 砸击<ref item="anvilcraft:piezoelectric_crystal"/>产生的电荷也会偏少
