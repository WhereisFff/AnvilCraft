# 鱼缸

<recipe id="anvilcraft:fish_tank"/>

# 铁砧加工

- 可以存储最多1000mB流体(一桶)，拥有8个原料格子容量、8个成品格子容量
- 持物品右击放入物品
- 代替炼药锅执行相关的铁砧合成
- 铁砧锤右键侧面可以将其变为有输出口的鱼缸，输出口自动输出加工后的物品

# 其他功能

<item id="minecraft:tropical_fish_bucket"/>

- 特殊的，持热带鱼桶/水桶右击鱼缸的下半部分，放入/取出热带鱼
- 可以戴在头上获得临时水下呼吸效果
