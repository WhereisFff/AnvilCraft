# 蟹笼

<recipe id="anvilcraft:crab_trap"/>

# 功能

- <ref item="anvilcraft:crab_trap"/>放置在水面，确保四面紧邻的方块至少三个为水源或含水方块
- 可以自动产出鱼获
- 不同生物群系产出略有不同，但都会产出<ref item="anvilcraft:crab_claw"/>

# 特性

右键或以铁砧砸之，可使其吐出产物。
