# 浮霜锻造台

<recipe id="anvilcraft:smithing/frost_smithing_table"/>

## 功能

- 不消耗任何锻造模板的特殊锻造台
- 通常需要配合<translate key="item.anvilcraft.permutation_smithing_template"/>或<translate key="item.anvilcraft.deformation_smithing_template"/>使用

# 嬗变锻造

<recipe id="anvilcraft:permutation_smithing_template"/>

保留物品的形态，转换其材质

- 消耗[<translate key="item.anvilcraft.royal_steel_ingot"/>](../002_material/110_royal_steel.md)，转化钻石武器工具 和 皇家钢武器工具
- 消耗[<translate key="item.anvilcraft.ember_metal_ingot"/>](../002_material/211_ember_metal.md)，转化下界合金武器工具 和 余烬金属武器工具
- 消耗[<translate key="item.anvilcraft.multiphase_matter"/>](../002_material/301_multiphase_matter.md)，转化浮霜金属武器工具 和 余烬金属武器工具
- 消耗[<translate key="block.anvilcraft.multiphase_matter_block"/>](../002_material/301_multiphase_matter.md)，转化浮霜铁砧、锻造台、砂轮 和 余烬铁砧、锻造台、砂轮
- 消耗[<translate key="block.anvilcraft.chromatic_stone"/>](../002_material/106_chromatic_stone.md)，转化四种[宝石护符](../006_prop/100_amulet.md)

# 形变锻造

<recipe id="anvilcraft:deformation_smithing_template"/>
保留物品的材质，转换其形态

消耗[<translate key="item.anvilcraft.frost_metal_ingot"/>](../002_material/202_frost_metal.md)使以下工具形态之间可以相互转换：
- 剑，斧，镐，锄，锹
- 头盔，胸甲，腿甲，靴子
- 弓，弩
