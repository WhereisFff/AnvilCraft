# 铁砧挖掘

# 定义

铁砧挖掘是一种概念。下述几种行为都实现了这个概念，因此拥有共同的性质

# 实现

- 通过[切石机配合铁砧](../006_struct/000_block_processing.md)破坏单个方块
- 通过[铁砧触发方块吞噬器](../005_block/101_block_devourer.md)破坏范围内方块
- 通过[巨型铁砧撼地](../005_block/210_giant_anvil.md)破坏大范围内方块

# 效果

使用不同的铁砧参与破坏方块的行为，具有不同的破坏效果：
