# 伽马激光

- 伽马机关是一种特殊的[激光](../004_block/201_basic_laser.md#激光)
- 来自<ref item="anvilcraft:celestial_forging_anvil"/>
- 最多传播16格
- 对生物造成的伤害是同等级普通激光的16倍
- 不能被<ref item="anvilcraft:ruby_prism"/>偏转，甚至会直接破坏<ref item="anvilcraft:ruby_prism"/>

# 加热

- 可以加热大范围<ref item="anvilcraft:ember_metal_block"/>至[超温状态](301_overheated_block.md)，
- 加热范围见下表

|   等级    |   范围    |
|:-------:|:-------:|
|  [4,7]  |  1个方块   |
| [8,11]  |  3x3方块  |
| [12,15] | 5x5x2方块 |
| [16,+∞) | 7x7x3方块 |

<info>
以7x7x3举例，指沿着激光方向穿透3层方块，并加热法向面积7x7
</info>

<tip>
可用于配合<ref item="anvilcraft:heat_collector"/>发电
</tip>

# 破坏方块

- 拥有凋灵的破坏能力，破坏沿途的一切透明与不透明方块
- 破坏**不产生**掉落物
- 破坏速度见下表

|   等级    | 破坏速度(个/秒) |
|:-------:|:---------:|
|  [4,7]  |    1/3    |
| [8,11]  |     1     |
| [12,15] |     4     |
| [16,+∞) |    20     |
