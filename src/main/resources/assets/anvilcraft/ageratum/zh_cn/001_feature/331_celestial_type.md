# 确定正确的参数组合

![star-info](../../textures/star_info.png)

1. 左、上、右、下四个轴依次表示天体的四个参数
2. 参数组会在左上、右上、左下构成3个焦点
3. 如果三个焦点颜色恰好相同，说明参数组成立，可以绑定到天体
4. 每个颜色都对应一类天体，表格如下
