# Void Decay

<row halign="center">
<item id="anvilcraft:void_matter_block"/>
</row>

# Definition

- Under certain conditions, <ref item="anvilcraft:void_matter_block"/> will spontaneously transform into random blocks

# Details

1. <ref item="anvilcraft:void_matter_block"/> has at least 5 faces touching <ref item="anvilcraft:void_matter_block"/>
2. Upon receiving a random tick, it transforms into one of the following blocks:
   - Various stone and sandstone from all three dimensions
   - Dirt, Coarse Dirt, Rooted Dirt
   - Mud, Clay, Ice
   - Iron, Copper, Gold Ores
   - Raw Iron Block, Oxidized Copper Block
   - Flint Block, Moss Block, Sculk Block
   - Nether Quartz Ore, Void Stone, End Dust
   - Zinc, Tin, Lead Ores
   - Various Infested Stone
