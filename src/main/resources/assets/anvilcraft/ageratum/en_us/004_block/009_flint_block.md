# Flint Block

<recipe id="anvilcraft:flint_block"/>

# Function

- Can be cut into building blocks via stonecutter
- When <ref item="minecraft:iron_block"/> and <ref item="anvilcraft:flint_block"/> rub against each other due to <ref item="minecraft:piston"/> movement,
  after the push, centered on the pushed block, ignites oil cauldrons and campfires in a 3*3*3 area; if neither exists, it will start a fire

# Properties

- Mineable with wooden pickaxe
- Drops itself when mined
