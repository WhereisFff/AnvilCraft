# Propel Piston

<recipe id="anvilcraft:propel_piston"/>

# Function

- When receiving a redstone signal or right-clicked by a player, continuously moves forward and pushes blocks
- Power consumption is proportional to the number of blocks being pushed
- Stops when manually right-clicked to stop, when energy is depleted, or when encountering an immovable block
- Charging:
  - Can be charged when the back receives a laser; charging power = Laser Level * 15kW
  - Can be manually charged using a capacitor
