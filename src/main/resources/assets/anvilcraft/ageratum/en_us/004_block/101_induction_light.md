# Induction Light

<recipe id="anvilcraft:induction_light"/>

# Function

- Provides illumination
- Right-clicking with different items sets the function mode; see the table below for details

|                  Corresponding Item                  | Working Range     | Function                      | Power Consumption (kW) |
|:---------------------------------------------------:|----------|-------------------------|:--------:|
|                          -                          | -        | -                       |    1     |
|    <ref item="minecraft:redstone"/>                 | 5x5x5 blocks | Accelerates crop growth, requires block light level of at least **10** |    16    |
| <ref item="minecraft:glowstone_dust"/>              | 3x3x3 subchunk | Suppresses **monster** spawning             |    16    |
|  <ref item="anvilcraft:void_matter"/>               | 3x3x3 subchunk | Suppresses **animal** spawning             |    16    |

<info>
Subchunk: A section of a **chunk**, 16x16x16 in size. Press F3+G to view, divided by blue lines
</info>

# Properties

- Cannot work when power is insufficient
