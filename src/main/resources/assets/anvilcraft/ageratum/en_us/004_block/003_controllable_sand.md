# Controllable Sand

<recipe id="anvilcraft:controllable_sand"/>

# Function

- <ref item="anvilcraft:controllable_sand"/> is not affected by gravity by default
- When receiving a redstone signal, it will **randomly** fall or rise
- If one direction is blocked by a block, it will definitely move in the opposite direction
- Can carry players while moving
