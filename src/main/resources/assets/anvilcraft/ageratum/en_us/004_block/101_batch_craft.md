## Function

- Idle power consumption: 4kW
- When receiving a redstone signal, immediately batch-crafts or stone-cuts up to 64 times
- Open the GUI to:
  - Set filters
  - Use the scroll wheel on slots to set item limits
- Shift+right-click with <ref item="minecraft:crafter"/> or <ref item="minecraft:stonecutter"/> in hand to switch modes
