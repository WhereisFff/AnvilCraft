# Frost Anvil

<recipe id="anvilcraft:smithing/frost_anvil"/>

## Function

- <ref item="anvilcraft:frost_anvil"/> never takes damage
- Has all the smithing capabilities of <ref item="anvilcraft:royal_anvil"/>
- When merging enchantments, the anvil penalty increases more slowly, +1 each time (vanilla is *2)
- Can consume frost metal to repair any tool without increasing anvil penalty
- Naming items costs no experience and does not increase anvil penalty
