# Transcendence Anvil

<recipe id="anvilcraft:smithing/transcendence_anvil"/>

## Function

- <ref item="anvilcraft:transcendence_anvil"/> never takes damage
- Has all the smithing capabilities of <ref item="anvilcraft:ember_anvil"/>
- When merging enchantments, **ignores** enchantment level caps; the anvil penalty increases more slowly, +1 each time
