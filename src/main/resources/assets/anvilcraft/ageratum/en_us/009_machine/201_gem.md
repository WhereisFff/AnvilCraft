# Processing: Gem Conversion Machine

- From reading [Gem Conversion](../008_recipe/204_gem.md), you understand its challenges
- Tossing in gem blocks, tossing in gems, swapping the work block under the cauldron, and striking the cauldron -- just thinking about it probably gives you a headache
- Let's open our minds and take a look at the example machine below; you will surely gain something from it

This machine continuously swaps the work block underneath by pushing and pulling the cauldron, while using <ref item="minecraft:smooth_stone"/> to strike the cauldron for processing. The complex and precise timing allows for very high processing speed.

<tip>
Left-click to adjust position; right-click to adjust angle; PgUp/PgDn to adjust display height
</tip>

1. Use a button to modify the copper bulb so it defaults to the lit state (you can remove the button afterwards)
2. Use the anvil hammer to right-click the front of the cauldron to create an output slot
3. The *dropper* facing the *comparator* should be filled with emerald blocks
4. The farther *dropper* should be filled with an equal amount of gems
5. Usage:
  - After loading materials, reset the lever to run
  - Pull the lever down to shut down

<warning>
Do not shut down during operation!
</warning>

- All <ref item="minecraft:smooth_stone"/> can be replaced with any full opaque block
- All <ref item="minecraft:copper_bulb"/> can be replaced with any copper bulb
- Ensure the <ref item="minecraft:observer"/> is facing the correct direction
