<recipe id="anvilcraft:stamping/pill"/>

# Pill Box

<recipe id="anvilcraft:pill_box"/>

## Storage

- Left-click to pick up a <ref item="anvilcraft:pill"/>, then left-click on the <ref item="anvilcraft:pill_box"/> to store it
- Hover the cursor over the <ref item="anvilcraft:pill_box"/>, use the scroll wheel to select a <ref item="anvilcraft:pill"/>, and right-click to take it out

## Usage

- Right-click to consume one of each different type of <ref item="anvilcraft:pill"/> inside at once, with a 2-second cooldown
- You can set a hotkey: <Key id="key.anvilcraft.use_pill_box"></Key> to use from your inventory (when there are multiple <ref item="anvilcraft:pill"/> in the inventory, the hotkey only uses the one closest to the front)
