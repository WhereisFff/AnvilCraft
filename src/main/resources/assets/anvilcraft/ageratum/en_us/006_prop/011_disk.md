# Disk

<recipe id="anvilcraft:disk"/>

## Function

- Right-click a device to copy its configuration
- If a configuration has already been copied, right-clicking another device of the same type pastes the configuration
- Shift + right-click to clear the configuration

<info>
Configuration does not include: block direction
</info>
