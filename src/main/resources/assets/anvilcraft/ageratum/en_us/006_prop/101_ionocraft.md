# Ionocraft

<row halign="center">
<recipe id="anvilcraft:ionocraft"/>
</row>

# Function

- Right-click to place on the ground, spawning an Ionocraft entity (similar to a boat entity; breaking it drops the Ionocraft item)
- Within a power grid range, it adds a 16kW load to the grid and ascends rapidly
- Outside a power grid range, it descends slowly
- Entities can stand on it
