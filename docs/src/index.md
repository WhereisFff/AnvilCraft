# 前言

欢迎来到《铁砧工艺》(AnvilCraft) 的官方文档！

## 什么是铁砧工艺？

铁砧工艺是一个以铁砧为核心的 Minecraft 原版生存拓展模组。该模组致力于在保持 Minecraft 原版风格的基础上，为玩家提供更多实用的机械和工艺内容，提升游戏体验。

## 核心功能

铁砧工艺提供了多种独特的工艺机制，包括但不限于：

* **磁铁系统**：利用磁铁将铁砧吸到空中，通过红石充能后释放，实现精准的物品处理
* **粉碎工艺**：将岩石等材料粉碎为更小的单位，如将圆石粉碎为沙砾
* **压榨系统**：从方块中提取液体，如从海绵中压榨出水
* **物品压缩与解压**：将零散物品压实成方块，或将方块分解为原材料
* **辊压工艺**：将金属材料压制成薄板，用于更高级的制作
* **膨发与晶化**：通过与炼药锅配合，实现材料的转化
* **压合工艺**：将两个方块压合成一个更高级的方块
* **方块破坏**：利用切石机精确破坏方块

## 设计理念

铁砧工艺的设计理念是"原版增强"，所有机制都围绕原版 Minecraft 的游戏风格展开：

1. **保持原版美学**：所有内容都遵循 Minecraft 原版的艺术风格和设计语言
2. **平衡性**：新机制不会破坏游戏平衡，而是为玩家提供更多选择
3. **易用性**：所有机制都直观易懂，符合玩家对 Minecraft 的预期
4. **实用性**：每个新功能都解决玩家在原版游戏中遇到的实际问题

## 适用人群

* 喜欢原版 Minecraft 生存玩法的玩家
* 希望获得轻量级自动化解决方案的玩家
* 对机械工艺感兴趣但不想使用复杂科技模组的玩家
* 想要扩展 Minecraft 原版机制的创作者和服务器管理员

## 技术信息

* **支持版本**：Minecraft 1.21 - 1.21.1
* **模组加载器**：支持 NeoForge 和 Forge
* **许可证**：代码采用 LGPL-3.0 许可证，资产采用 ARR 许可证

## 开始使用

要开始使用铁砧工艺，只需下载对应版本的 mod 文件并放入游戏的 `mods` 文件夹中即可。无需额外配置，模组会自动集成到游戏中。
